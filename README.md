# Finance Dashboard - Internship Assignment 🚀

## 🎯 Overview
Interactive Angular 21 + Tailwind dashboard for financial tracking with:
- **Dashboard Overview** - Summary cards, balance trend, **clickable spending pie**
- **Transactions** - CRUD + search/filter/sort
- **Insights** - **Clickable charts + category table**
- **GLOBAL FILTER** - Click chart legend/table → ALL pages filter instantly! 🔄
- Role-based UI (Admin/Viewer), Light/Dark theme, Fully responsive

## 🚀 Quick Start
```bash
cd finance-dashboard
npm install
ng serve
```
Open http://localhost:4200
